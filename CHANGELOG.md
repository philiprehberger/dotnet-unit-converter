# Changelog

All notable changes to this project will be documented in this file.

## [0.1.0] - 2026-03-21

### Added

- Initial release.
- Type-safe unit conversions with fluent API (`Convert.From(...).To(...)`).
- Formatted output with `Convert.Format(...)`.
- Length units: Meters, Kilometers, Centimeters, Millimeters, Inches, Feet, Yards, Miles.
- Weight units: Kilograms, Grams, Milligrams, Pounds, Ounces, Tons.
- Temperature units: Celsius, Fahrenheit, Kelvin.
- Volume units: Liters, Milliliters, Gallons, Quarts, Cups, FluidOunces.
- Data size units: Bytes, KB, MB, GB, TB, PB.
- Speed units: MetersPerSecond, KilometersPerHour, MilesPerHour, Knots.
